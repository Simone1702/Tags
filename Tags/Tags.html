O que são as tags "h1 a h6"


(Voltar)

As tags "h" são usadas como título e subtítulo na criação do conteúdo da página html. Existem seis tags "h" para ser usadas, começando pela tag h1 que representa o título principal do conteúdo da página. As tag seguem uma ordem decrescente ao serem utilizadas. A seguir veremos os exemplos:

A tag h1

usada como título principal.

A tag h2

usada como uns do subtítulo na criação de uma página.

A tag h3

também usada como um subtítulo.

A tag h4

usada como subtítulo.
A tag h5
usada como subtítulo.
A tag h6
usada como subtítulo.

Apesar de conter seis tags para uso título normalmente é usado apenas as tags h1, h2, h3 ou apenas a tag h1 e h2.

O que são as tags "p"


(Voltar)

As tags "p" são usadas para incluir os textos dentro do documento html. As tags "p" são conhecidas como tags parágrafo.

O que são as tags "small, mark"

(Voltar)

A tag "small" é usada para deixar o texto menor do restante do texto ou frase.

A tag "mark" serve para marcar determinada parte do texto palavra ou frase. a tag "mark" serve como um tipo de marca texto

O que são as tags "i, u, strong"


(Voltar)

Agora vamos falar das tags,"i" que serve para deixar o texto em itálico. Por serem tags podemos utilizar elas em uma palavra apenas, ou frase. Essa regra se aplica a qualquer tag que deseje usar na criação do seu conteúdo desde que ela faça sentido ao que está sendo criado.

Já as tags "u" são usadas para sublinhar alguma parte do texto ou frase.

A tag "strong" serve para deixar um texto em negrito quando queremoss chamar a atenção para aquela frase ou palavra queremos deixar em destaque.

O que são as tags "ol, ul, li, a"

(Voltar)

As tags "ol, ul, li." são utilizadas na criação de diferente tipos de listas.

A tag "ol" é uma tag de lista ordenada, porem para que ela funcione de forma correta ela precisa de sua tag filha. Mas o que são tags filhas?.

No caso a tag "ol" é uma tag "pai" ou podemos dizer de forma simples uma tag principal que precisa de uma outra tag secundaria para poder ser usada de forma correta.

Mas e a tag "ul" e "li"?

Vamos falar da tag "li". Essa tag é considerada uma tag filha pois é ela que vai dar corpo e sentido a tag "ol". Que é a tag que lista os item de uma lista, indice, ou menu.

Já a tag "ul" significa lista não ordenada, ou seja ela segue uma ordem especifica.

Vamos obrservar alguns exemplos de criação de listas com a tag "ul, ol, li".

  1. Tag ol 1.
  2. Tag ol 2.
  3. Tag ol. 3
  • Tag ul 1.
  • Tag ul 2.
  • Tag ul. 3

No exemplo acima observa-se que a tag "ol' organiza a lista de forma numérica.

Já a tag "ul" não utiliza um sequencia numérica ordenada para criar a lista.

A tag "li" representa os itens listados ou melhor os itens marcador em amarelo.

Mas a tag "a" o que ela faz? A tag "a" é uma tag de ancoragem ou seja ela quem vai tornar nossa página cliclavel. Para que a tag "a" funcione da forma que desejamos devemos incliuir o atributo ' href="" ' que é onde vamos colocar o link de uma outras página html, site, blog, etc...

O que são as tags "font, del, abbr, input, footer, inframe, link, video."

(Voltar)

Agora iremos falar da tag "font", essa tag é usada para realizar alteração dos caracteres de uma palavra, texto, ou frase para que a tag funcione de forma correta devemos incluir o atributo face="" a ela. Ficando da seguinte forma '< font face="verdana">< /font>' . Assim sendo possivel alterar o estilo de fonte padrão, dentro do próprio html.

Enquanto que a tag "del" é usada para definir um texto que foi excluído do conteúdo apresentado ou seja o texto aparecera de forma riscadano texto, ou frase.

A tag "abbr" é uma tag de abreviação, que possuí significado semantico alem da sua importancia para os mecanismo de busca da web. Um exmplo que podemos usar a tag "abbr" seria quando queremos informar um estado de forma abreviada, ficando da seguinte forma SP. Para o usuário a tag mostra no texto apenas a sigla SP, porém ao navegafor a tag fica seguinte forma '< abbr title="São Paulo">SP < /abbr>, dentro da tag "abbr" fica o atributo title que irá indicar ao navegador que a abreviação de SP é a palavra SÃO PAULO.
